# Polyv WebView SDK Demo

## 1 项目介绍

本项目从属于广州易方信息科技股份有限公司，是一款基于PLVWebViewSDK，为客户提供快速集成 WebView 控件到应用中的 Demo。PLVWebViewSDK 对保利威云直播产品的直播、回放观看做了良好的适配，极大优化了用户的观看体验，并支持浮窗播放等扩展功能，也可作为其它网页的展示容器。

本项目包含功能如下：

- 支持基础WebView功能
- 支持应用内浮窗播放
- 支持悬浮窗手势拖动
- 支持配置悬浮窗大小
- 支持配置悬浮窗初始位置
- 支持视频全屏播放
- 支持系统小窗播放
- 支持后台自动开启系统小窗
- 支持WebView Feed流播放
